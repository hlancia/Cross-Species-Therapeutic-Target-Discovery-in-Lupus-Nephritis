# Scripts

This directory contains the complete reproducible R workflow used for cross-species therapeutic target discovery in lupus nephritis.

The analysis is organized into five sequential scripts that progress from human single-cell RNA sequencing analysis to the final consensus ranking of therapeutic targets.

---

## Analysis Pipeline

| Script | Purpose | Main Output |
|---------|---------|-------------|
| **01_human_discovery.R** | Preprocesses the human AMP Lupus Nephritis single-cell RNA sequencing dataset, performs quality control, clustering, and cell type annotation. | Human cell atlas, annotated clusters |
| **02_target_prioritization.R** | Identifies inflammatory macrophage marker genes and prioritizes candidate therapeutic targets using differential expression analysis. | Candidate target lists and marker tables |
| **03_mouse_validation.R** | Evaluates candidate targets in an independent mouse lupus nephritis single-cell RNA sequencing dataset to assess cross-species conservation. | Cross-species validation results |
| **04_spatial_validation.R** | Validates prioritized targets using an independent human spatial transcriptomics dataset and evaluates their localization within inflamed kidney tissue. | Spatial validation results |
| **05_consensus_ranking.R** | Integrates evidence from all datasets to generate the final consensus ranking of therapeutic targets and produces the summary figures. | Final target rankings and consensus figures |

## Reproducibility

The scripts are intended to be executed sequentially. Each script uses the outputs generated by the previous stage of the analysis.

Running the complete workflow reproduces:

- Human cell atlas
- Differential expression analyses
- Candidate therapeutic target lists
- Cross-species validation
- Human spatial transcriptomic validation
- Final consensus therapeutic target ranking
- Publication-quality figures
